# Valentine Invitation Website for Koco

## Original Problem Statement
Build a single-page Valentine invitation website for girlfriend "Koco" with:
- Animated teddy bear landing with envelope
- Interactive envelope opening animation
- Yes/No buttons (No button dodges)
- Confetti celebration on Yes
- Save screenshot feature
- Mobile-first responsive design
- Accessibility support (prefers-reduced-motion)

## User Persona
- Target: Girlfriend named "Koco"
- Context: Romantic Valentine's gift
- Goal: Interactive, cute, premium experience

## Core Requirements (Static)
- Soft pastel pinks/creams color palette
- Script font for headings (Dancing Script)
- Clean font for body (Nunito)
- Mobile-first design (iPhone sizes)
- Accessibility features
